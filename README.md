# Setup
* 'bundle install' - use bundler to install all needed gems defined in Gemfile
* 'jekyll build'

## Deploy to ftp server
* change url in `_config.yml`
* ftp settings in `ftp_configuration.yml`
* execute `thor ftp:deploy`
